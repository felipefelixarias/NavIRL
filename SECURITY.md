# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability, please report it privately when
possible. If GitHub Security Advisories are enabled for this repository, open
one with details and reproduction steps.

If private reporting is not available, open a minimal public issue that does
not disclose sensitive details and request a private channel.

## Supported Versions
Only the latest minor release is actively supported. Please upgrade to the
most recent version before reporting.
